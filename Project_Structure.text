What we are building in this Chatbot
Core RAG Engine - Upload menu PDF, process it, query it
Streamlit Web Interface - Clean chat UI for demo/testing
WhatsApp Integration Ready - Structure that you can plug WhatsApp API into
Price filtering, dietary preferences, smart suggestions
